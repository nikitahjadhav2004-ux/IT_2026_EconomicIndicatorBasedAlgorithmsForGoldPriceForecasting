# Economic Indicator-Based Algorithms for Gold Price Forecasting

## Project Description
This project predicts gold prices using Deep learning models based on economic indicators such as Gold prices, CPIndex, USD_INR rate.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Files Included
- lstm.ipynb
- randomforrestregressor.ipynb
- xgboost.ipynb
- CPIndex_Jan15-To-Jan25.csv
- Gold Price.csv
- USD_INR Historical Data.csv

## How to Run
1. Open notebook in Google Colab
2. Upload dataset files
3. Run all cells

## Output
Gold price predictions and accuracy comparison of models.
